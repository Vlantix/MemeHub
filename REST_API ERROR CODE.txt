📊 REST API Error Codes
4xx Client Errors (Frontend's fault)
Code	Name	When to Use	Example Response
400	Bad Request	Missing or invalid data format	{"error": "All fields are required"}

401	Unauthorized	Not logged in / invalid token	{"error": "Invalid credentials"}

403	Forbidden	Logged in but no permission	{"error": "Access denied"}

404	Not Found	Resource doesn't exist	{"error": "User not found"}

409	Conflict	Username/email already exists	{"error": "Username is already taken"}

422	Unprocessable Entity	Validation fails (password too short)	{"error": "Password must be at least 6 characters"}

429	Too Many Requests	Rate limiting (too many login attempts)	{"error": "Too many attempts. Try again later"}


5xx Server Errors (Backend's fault)
Code	Name	When to Use	Example Response
500	Internal Server Error	Database connection fails, unhandled exception	{"error": "Something went wrong"}

503	Service Unavailable	Database down, server overloaded	{"error": "Service temporarily unavailable"}
